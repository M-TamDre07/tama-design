Jawaban Jujur

Tidak ada di sini? Tanya langsung ke AI Assistant saya di bawah!

Berapa kisaran harga desain?

Harga bervariasi tergantung jenis dan kompleksitas desain. Sebagai gambaran: Poster & Flyer (Rp 50rb–150rb), Banner (Rp 75rb–250rb), Konten Sosial (Rp 60rb–200rb), Logo (Rp 200rb–600rb), Presentasi (Rp 100rb–400rb), Edit Foto (Rp 50rb–200rb). Untuk estimasi akurat, konsultasi gratis via WA.

Berapa lama pengerjaan desain?

1–3 hari kerja untuk desain sederhana (poster, konten sosial). Untuk proyek kompleks seperti logo atau presentasi: 3–7 hari. Butuh cepat? Beritahu di awal, ada layanan ekspres.

Apakah bisa revisi desain?

Tentu! Revisi dilakukan hingga desain sesuai keinginan Anda. Paket standar: 2–3 revisi. Butuh revisi lebih banyak? Bisa didiskusikan dengan paket custom.

Format file apa yang akan saya terima?

Format JPG (resolusi tinggi), PNG (background transparan jika diperlukan), atau PDF (print-ready). File source (.AI, .PSD) tersedia dengan biaya tambahan.

Bagaimana proses pemesanan?

Mudah! Isi form di bawah atau chat WhatsApp langsung. Setelah diskusi konsep dan kesepakatan, Anda bayar, dan saya mulai kerjakan. Progress diupdate secara berkala.

Apakah ada paket bundling?

Ada! Paket sosial media bulanan, paket brand starter (logo + banner + konten), dan paket custom sesuai kebutuhan. Hubungi via WA untuk info harga dan detail paket.

Apa itu layanan konsultasi desain?

Layanan konsultasi gratis di mana Anda bisa berdiskusi tentang strategi visual brand, arah desain yang tepat, referensi, dan apa saja yang dibutuhkan — sebelum memutuskan untuk memesan. Kunjungi konsultasidesignbytamaandrea.vercel.app.
