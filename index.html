FAQ

Pertanyaan yang Sering Diajukan

Tidak menemukan jawaban? Tanyakan langsung ke AI Assistant di bawah.

Berapa lama pengerjaan desain?

1–3 hari kerja untuk desain sederhana (poster, konten sosial). Untuk proyek kompleks seperti logo atau presentasi: 3–7 hari. Butuh cepat? Beritahu di awal, ada layanan ekspres.

Apakah bisa revisi desain?

Tentu! Paket standar mencakup 2–3 revisi minor. Butuh revisi lebih banyak atau perubahan mayor? Bisa didiskusikan dalam paket custom.

Format file apa yang akan saya terima?

Format JPG (resolusi tinggi), PNG (background transparan jika diperlukan), atau PDF (print-ready). File source (.AI, .PSD) tersedia dengan biaya tambahan.

Bagaimana proses pemesanan?

Mudah! Konsultasi dulu melalui tautan di website ini atau chat WhatsApp. Setelah diskusi konsep dan kesepakatan harga, Anda bayar, dan saya mulai kerjakan.

Apakah ada paket bundling?

Ada! Paket sosial media bulanan, paket brand starter (logo + banner + konten), dan paket custom. Hubungi saya untuk penawaran khusus.

Mengapa harganya terlihat lebih tinggi dari sebelumnya?

Karena kami berkomitmen pada kualitas profesional. Setiap desain dikerjakan dengan standar tinggi, revisi inklusif, dan file resolusi penuh — bukan sekadar template murah. Investasi yang tepat untuk brand Anda.
