Tanya Jawab

Mungkin pertanyaan Anda ada di sini. Jika tidak, tanya AI assistant di bawah!

Berapa lama pengerjaan desain?

Estimasi pengerjaan tergantung tingkat kerumitan desain. Umumnya 1-3 hari kerja untuk desain sederhana.

Apakah saya bisa revisi desain?

Tentu, revisi bisa dilakukan sampai desain sesuai dengan keinginan Anda. Jumlah revisi tergantung paket yang dipilih.

Format file apa yang akan saya terima?

Anda akan mendapatkan file final dalam format JPG, PNG, atau PDF. Jika diperlukan, saya juga bisa menyediakan file mentah seperti .AI atau .PSD dengan biaya tambahan.

Software apa saja yang saya gunakan?

Saat ini saya sudah menguasai Canva Pro dan ibis Paint. Sedang dalam proses pembelajaran Adobe Illustrator, Photoshop, dan Figma.
